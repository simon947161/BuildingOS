# BuildingOS

BuildingOS is the official local workspace for BuildingOS specification,
governance, roadmap, and architecture documents.

This workspace normalizes the previous BuildingOS work that was developed in
the old `buildingos-modular-interface` working folder.

## Official Repository

```text
https://github.com/simon947161/BuildingOS.git
```

## Current Scope

This workspace is documentation-first.

It currently preserves:

- BuildingOS M1 specification foundation documents;
- glossary, evidence, claim, identity, and review standard drafts;
- freeze records;
- engineering workflow and incremental review governance;
- roadmap, core architecture, and engineering plan records.

No runtime implementation, Python package, SQLite schema, API, MCP tool, or PRI
implementation is introduced by workspace normalization.
